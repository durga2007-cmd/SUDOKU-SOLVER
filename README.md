🧩 Sudoku Solver (Backtracking Algorithm)
📌 Overview

This project is a web-based Sudoku Solver that solves 9×9 Sudoku puzzles using the backtracking algorithm. It provides an interactive interface for users to input puzzles and instantly get solutions.

This project supports SDG 4 – Quality Education by helping students understand logical thinking and algorithmic problem-solving.

🚀 Features
9×9 interactive Sudoku grid
Solve Sudoku using backtracking algorithm
Input validation (row, column, 3×3 box rules)
Clear/reset grid option
Generate random Sudoku puzzle (optional)
Fast and accurate solution

🛠️ Technologies Used
HTML
CSS
JavaScript

🧠 How It Works

The solver uses the backtracking algorithm:

Find an empty cell
Try numbers from 1 to 9
Check if the number is valid:
Not present in the same row
Not present in the same column
Not present in the same 3×3 box
If valid → move to next cell
If invalid → backtrack and try another number
Repeat until the puzzle is solved

▶️ How to Run
Download or copy the project files
Save as sudoku.html
Open the file in any web browser
Enter numbers in the grid
Click Solve to get the solution

🧪 Usage
Enter a valid Sudoku puzzle
Click Solve
The solved puzzle will be displayed instantly
Use Clear to reset the board

❗ Important Notes
Only valid Sudoku puzzles can be solved
If the input is incorrect, the app will show:
"No solution exists for this puzzle"
